# AcadSoft - Python Scripts

Pasta contendo scripts Python para:
- 📊 Análise de dados
- 📈 Geração de relatórios
- 🔄 Automação de tarefas
- 🔗 Integração com APIs externas

## Como usar

1. Instalar dependências:
```bash
pip install -r requirements.txt
```

2. Executar scripts:
```bash
python seu_script.py
```

## Scripts disponíveis

### Relatórios
- `relatorio_alunos.py` - Gera relatório de alunos
- `relatorio_instrutores.py` - Gera relatório de instrutores
- `relatorio_agendamentos.py` - Gera relatório de agendamentos

### Análise
- `analise_dados.py` - Análise avançada de dados

### Automação
- `enviar_lembretes.py` - Envia lembretes de aulas
- `backup.py` - Backup de dados

Adicione seus scripts nesta pasta!
